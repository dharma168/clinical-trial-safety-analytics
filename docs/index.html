🧪 Clinical Trial Safety & Risk Analytics Project

📌 Overview

End-to-end Clinical Data Management (CDM) and analytics project using SDTM-style datasets from a prostate cancer clinical trial.

🎯 Objective

  • Clean and structure clinical trial data
  • Perform patient-level integration
  • Analyze adverse events and lab data
  • Build risk stratification model
  • Create Power BI dashboard

📊 Tools Used

  • Python (Pandas)
  • SQL
  • Microsoft Power BI
  • Jupyter Notebook

🔄 Workflow

  1. Data Cleaning & Preprocessing
  2. SDTM-style structuring
  3. Patient-level merging (DM, AE, LB)
  4. Risk model creation
  5. SQL analysis
  6. Dashboard visualization

📈 Key Insights

  • High AE patients show higher lab burden
  • Risk model successfully identifies critical patients
  • Integrated dataset improves clinical interpretation
