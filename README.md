# Clinical Trial SDTM Project 

Project Title

Clinical Trial SDTM-Based Analysis (Prostate Cancer Study)

Objective

To analyze clinical trial data (Demographics (DM), Adverse Events (AE), Laboratory (LB), Treatment Exposor (EX)) and generate patient-level safety and risk insights.

Workflow
Data cleaning and decoding
SDTM-style structuring
Patient-level aggregation
Integration of AE + LB + DM
Risk stratification model

Key Outputs
Patient-level clinical dataset
Risk categories (Low / High)
AE and lab burden analysis
Visual insights

Key Insight
High AE patients show higher lab burden correlation
Exposure dataset was limited and excluded from population analysis
